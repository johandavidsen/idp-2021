Object

Et objekt (object) er en måde at binde tilhørende data sammen.

I JS siger vi at et object har egenskaber (properties) med hver sin værdig og type.

var obj= {
    a: "hello world",
    b: 42,
    c: true
};

typeof obj.a; // "hello world"
typeof obj.b; // 42
typeof obj.c; // true

Object

Et object kan forstås på samme måde som et array, forskellen er at et array kun kan tilgåes med et tal og et object kan tilgåes med propeties navn.

a b c
"hello World" 42 true

Adgang til egenskaberne

Du kan tilgå egneskaberne til et object på 2 forskellige måder.

Et object fungere på samme måde som et array nå man skal tilgå properties.

// DOT notation
obj.a // "hello world"

// Array access
obj["a"] // "hello world"

// Special case
obj["hello world!"]

Hvad er JavaScript?

JavaScript er et scripting/programmerings sprog som blev udviklet av Brendan Eich i 1995. Dengang arbejde Brendan Eich hos Netscape. Siden 1995 er sproget løbende blevet udviklet og forbedret.

Sprogret blev udviklet til at blive brugt på hjemmesider. Formålet med JavaScript er at det gøre det muligt at have interaktive aktioner, som for eksempel kort, 2D/3D grafik, video og musik afspillere på hjemmesider.

Sammen med HTML og CSS danner JavaScript grundsøjlerne for det moderne internet.

Hvad er JavaScript?

  • HTML

    HTML er det markup sprog som vi bruger til at give mening til vores hjemmeside indhold. Eksempler på disse er paragrafer, overskrifter, tabeller, billeder, video med mere.

  • CSS

    CSS er et style sprog som vi bruger til at give HTML det udseende som vi ønsker. Eksempler kan være baggrundsfarver eller størrelse på skriften.

  • JavaScript

    JavaScript er så scripting/programmering som vi bruger til at give live til HTML. Så som at opdatere dynamisk indhold med mere.

Hvad er JavaScript?

Historien gøre at disse 3 teknologier (HTML, CSS og JS) er knyttet sammen. Det vil så sige, at en dygtig JavaScript programmør mestrer både HTML, CSS og JS.

Selv om disse sprog som udgangspunkt danner grundsøjlerne for det moderne internet, kan de også bruges til at lave desktop, server og mobil applikationer (mere om det senere).

JavaScript og WebAssembly er de eneste sprog, som kan køres i en browser.

Hvad gør at JavaScript fungerer?

Som vi har været inde på tidligere, så er JavaScript et sprog, og for at JavaScript kan køre skal vi have noget der kan læse og køre sproget. Dette kalder vi en engine eller motor.

Der findes flere forskellige motorer og de er gerne vedligeholdt, som en del af en browser.

  • V8

    Denne bliver brugt i Google Chrome og i Microsoft Edge

  • SpiderMonkey

    Denne bliver brugt i Mozilla Firefox

  • JavaScriptCore

    Denne bliver brugt i Apple Safari

  • Chakra

    Denne blev brugt i Microsoft Internet Explorer, men bliver ikke vedligeholdt længere.

Hvad er så Nodejs?

Nodejs er en implementation of V8 der kan køre på en komputer eller server. Dette bliver også kaldt server-side.

Første udgave af Nodejs blev lavet i 2009. Styrken med Nodejs, sammenlignet med andre programmerings sprog var, at for første gang var det mulig for en udvikler at lave webapps i et sprog, både client-side og server-side.

Hvad er så Nodejs?

Som en naturlig complement til Nodejs blev NPM lavet. NPM er en pakke-manager, som gør det muligt at opdele en applikation i moduler og så at genbruge disse moduler i flere applikationer.

Der udover giver Nodejs adgang til nogle server moduler, som ikke er tilgængelige i browseren.
