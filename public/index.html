Object

Et objekt (object) er en måde at binde tilhørende data sammen.

I JS siger vi at et object har egenskaber (properties) med hver sin værdig og type.

var obj= {
    a: "hello world",
    b: 42,
    c: true
};

typeof obj.a; // "hello world"
typeof obj.b; // 42
typeof obj.c; // true

Object

Et object kan forstås på samme måde som et array, forskellen er at et array kun kan tilgåes med et tal og et object kan tilgåes med propeties navn.

a b c
"hello World" 42 true

Adgang til egenskaberne

Du kan tilgå egneskaberne til et object på 2 forskellige måder.

Et object fungere på samme måde som et array nå man skal tilgå properties.

// DOT notation
obj.a // "hello world"

// Array access
obj["a"] // "hello world"

// Special case
obj["hello world!"]
