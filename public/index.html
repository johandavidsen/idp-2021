Datalogi historie

Formålet er at gennemgå datalogiens historie.

Oprindelsen (2000 BC)

Det er svært at sig hvornår historien begynder, men et tidligt eksempel på datalogi er kuglerammen.

Pascaline (1642 AD)

Blaise Pascal opfandt i 1642 AD en mekanisk regnemaskine.

Leibniz Mekaniske regnemaskine (1674)

I 1674 lavede tyskeren Gottfried Wilhelm Leibniz en maskine, som kunne addere, subtrahere, dividere og multiplisere.

Jacquards væv (1801)

I 1801 lavede Joseph Marie Jacquard en mekanisk væv som kunne programmeres til at væve forskellige mønstre.

Analytiske Maskine (1837)

I 1837 udtænkte Charles Babbage en maskine som kunne programmeres (generelt anvendelige), den var digital, havde hukommelse og håndterede input og output. Denne maskine havde på denne måde alle de grundlæggende elementer som man finder i dagens computere.

Elektronrør (1907)

I 1907 blev de første elektronrør fremstillet som gjorde det muligt at styre strøm. Dette dannede så grundlaget for at man kunne lave logiske gates, så som AND, OR, NOR og NAND.

Flip-Flop (1918)

I 1918 lavede briterne William Eccles og F. W. Jordan et kredsløb som kunne have 2 stabile tilstande. Disse tilstande styres af et eller flere styresignaler. Almindeligvis assosierer vi disse 2 tilstande til 0 og 1. Dette var begyndelse for der som vi i dag kalder for hukommelse.

Tidsperioden fra 1918 til 1941

Det tog do noget tid før man for alvor begyndte på de digitalemaskiner og i perioden fra 1918 til 1941 bleve der stadig lavet mekaniske beregningsmaskiner.

Konrad Zuse’s Z3 (1941)

I 1941 lavede Konrad Zuse den første generelt anvendelige computer. Denne computer var delvist elektronmekanisk og i 1948 var den fuldt elektronisk, hvor man emmede programmet i hukommelsen.

Electronic Numerical Integrator and Computer (ENIAC) (1945)

ENIAC blev lavet i 1945 og havde i udgangspunktet de samme egenskaber som Z3. Den blev finansieret af den Amerikanske Hær, men blev dedikeret til University of Pennsylvania i 1946.

Manchester Baby (1948)

I 1948 lavede man på University of Manchester, det som er kendt som Manchester Baby. Som er et andet eksempel på elektronisk computer med indbyggede programmer. Disse maskiner var kæmpe store fordi at elektronrørene var forholdvis store.

Transistor (1947)

I 1947 blev den første transistor lavet af John Bardeen og Walter Brattain imens de arbejde under William Shockley hos Bell Labs. Oprindeligt var den fysikeren Julius Edgar Lilienfeld introducerede transistoren i 1926, men det var ikke muligt på det tidspunkt at lave den. Bardeen, Brattain og Shockley fik Nobel pris for opfindelsen i 1956. Transistoren gjorde den muligt at lave minder og billigere radio, regnemaskiner og computere.

Integreret kredsløb (1958)

Før 1958 var der forskellige forslag om at man samlede flere elektroniske komponenter på et enkelt kredsløb, men den store udfordring var at elektrisk isolere komponenterne sådan at de ikke forstyrrede hinanden. Dette blev først gjort muligt i 1958 af Jack Kilby der arbejde på Texas Instrument. Der var andre der arbejde på denne udfordring omkring samme tidspunkt der iblandt Robert Noyce som arbejde hos Fairchild Semiconductor. Senere var Noyce med til at starte Intel sammen med Gordon Moore.

Object

Et objekt (object) er en måde at binde tilhørende data sammen.

I JS siger vi at et object har egenskaber (properties) med hver sin værdig og type.

var obj= {
    a: "hello world",
    b: 42,
    c: true
};

typeof obj.a; // "hello world"
typeof obj.b; // 42
typeof obj.c; // true

Object

Et object kan forstås på samme måde som et array, forskellen er at et array kun kan tilgåes med et tal og et object kan tilgåes med propeties navn.

a b c
"hello World" 42 true

Adgang til egenskaberne

Du kan tilgå egneskaberne til et object på 2 forskellige måder.

Et object fungere på samme måde som et array nå man skal tilgå properties. 

// DOT notation
obj.a // "hello world"

// Array access
obj["a"] // "hello world"

// Special case
obj["hello world!"]

Hvad er JavaScript?

JavaScript er et scripting/programmerings sprog som blev udviklet av Brendan Eich i 1995. Dengang arbejde Brendan Eich hos Netscape. Siden 1995 er sproget løbende blevet udviklet og forbedret.

Sprogret blev udviklet til at blive brugt på hjemmesider. Formålet med JavaScript er at det gøre det muligt at have interaktive aktioner, som for eksempel kort, 2D/3D grafik, video og musik afspillere på hjemmesider.

Sammen med HTML og CSS danner JavaScript grundsøjlerne for det moderne internet.

Hvad er JavaScript?

  • HTML

    HTML er det markup sprog som vi bruger til at give mening til vores hjemmeside indhold. Eksempler på disse er paragrafer, overskrifter, tabeller, billeder, video med mere.

  • CSS

    CSS er et style sprog som vi bruger til at give HTML det udseende som vi ønsker. Eksempler kan være baggrundsfarver eller størrelse på skriften.

  • JavaScript

    JavaScript er så scripting/programmering som vi bruger til at give live til HTML. Så som at opdatere dynamisk indhold med mere.

Hvad er JavaScript?

Historien gøre at disse 3 teknologier (HTML, CSS og JS) er knyttet sammen. Det vil så sige, at en dygtig JavaScript programmør mestrer både HTML, CSS og JS.

Selv om disse sprog som udgangspunkt danner grundsøjlerne for det moderne internet, kan de også bruges til at lave desktop, server og mobil applikationer (mere om det senere).

JavaScript og WebAssembly er de eneste sprog, som kan køres i en browser.

Hvad gør at JavaScript fungerer?

Som vi har været inde på tidligere, så er JavaScript et sprog, og for at JavaScript kan køre skal vi have noget der kan læse og køre sproget. Dette kalder vi en engine eller motor.

Der findes flere forskellige motorer og de er gerne vedligeholdt, som en del af en browser.

  • V8

    Denne bliver brugt i Google Chrome og i Microsoft Edge

  • SpiderMonkey

    Denne bliver brugt i Mozilla Firefox

  • JavaScriptCore

    Denne bliver brugt i Apple Safari

  • Chakra

    Denne blev brugt i Microsoft Internet Explorer, men bliver ikke vedligeholdt længere.

Hvad er så Nodejs?

Nodejs er en implementation of V8 der kan køre på en komputer eller server. Dette bliver også kaldt server-side.

Første udgave af Nodejs blev lavet i 2009. Styrken med Nodejs, sammenlignet med andre programmerings sprog var, at for første gang var det mulig for en udvikler at lave webapps i et sprog, både client-side og server-side.

Hvad er så Nodejs?

Som en naturlig complement til Nodejs blev NPM lavet. NPM er en pakke-manager, som gør det muligt at opdele en applikation i moduler og så at genbruge disse moduler i flere applikationer.

Der udover giver Nodejs adgang til nogle server moduler, som ikke er tilgængelige i browseren.

Hvad er så forskellen på Browser og Nodejs?

Den helt store forskelle på at køre JavaScript i browseren og i Nodejs er de moduler, der er tilgængelige. Dette kalder vi også for browser API eller Nodejs API. API er en forkortelse for Application Programming Interface.
